# 🧺 বই এর কোট্টা (Boi Kotta) 
### "জ্ঞানের ঝুড়ি ভরুক সাশ্রয়ে"

বই এর কোট্টা হলো একটি আধুনিক ও সাশ্রয়ী প্ল্যাটফর্ম, যার লক্ষ্য বইপ্রেমীদের কাছে সহজে এবং সুলভ মূল্যে বই পৌঁছে দেওয়া। এটি আমাদের টিমের একটি সমন্বিত উদ্যোগ।

---

## ✨ বৈশিষ্ট্যসমূহ (Features)
* 🎯 **প্রিমিয়াম ডার্ক থিম:** চোখের আরামদায়ক ও আধুনিক ইউজার ইন্টারফেস।
* 🛡️ **অ্যাডমিন ড্যাশবোর্ড:** সিকিউর লগইন সিস্টেমের মাধ্যমে টিমের অভ্যন্তরীণ তথ্য ব্যবস্থাপনা।
* 📱 **ফুললি রেসপন্সিভ:** মোবাইল, ট্যাবলেট এবং পিসি—সব ডিভাইসেই পারফেক্ট দেখাবে।
* 🤝 **টিম ম্যানেজমেন্ট:** পাবলিকলি মেম্বারদের নাম এবং প্রাইভেটলি বিস্তারিত তথ্য দেখার সুবিধা।
